# Pagination Project

A simple and responsive Pagination application built using **HTML, CSS, and JavaScript**. This project demonstrates how to dynamically display content cards and navigate between them using Previous and Next buttons.

## 🚀 Live Demo

👉 https://vaibhavsain.github.io/Pagination/

## 📌 Features

* Dynamic course card rendering
* Previous and Next page navigation
* Responsive design
* DOM Manipulation using JavaScript
* Efficient pagination logic
* Clean and beginner-friendly code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 How It Works

1. Course data is stored in a JavaScript array.
2. Two cards are displayed per page.
3. Clicking **Next** loads the next set of courses.
4. Clicking **Previous** loads the previous set of courses.
5. The UI updates dynamically without reloading the page.
